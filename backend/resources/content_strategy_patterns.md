Content Strategy Patterns

- Objective-first planning:
  1) Objective
  2) Audience
  3) Tone
  4) Platform
  5) Success metric

- Narrative pattern for short promos:
  - Hook
  - Value proof
  - Emotional payoff
  - CTA

- Interests mapping:
  - If user interest includes education, favor clarity and examples.
  - If user interest includes gaming, favor energetic and bold language.
  - If user interest includes lifestyle, favor aspirational visual storytelling.
